# 🛒 Amazon Clone - Full Stack E-Commerce Application

A full-stack e-commerce web application inspired by Amazon, built using React, Node.js, Express, PostgreSQL, and JWT. The application allows users to browse products, manage their cart, place orders, and provides role-based admin functionality for product management.

## 🚀 Features

### Authentication & Authorization

* User Registration
* User Login
* JWT Authentication
* Protected Routes
* Role-Based Authorization (Admin/User)
* Secure Logout

### Product Management

* View All Products
* Search Products
* Product Details Page
* Add Products (Admin Only)
* Edit Products (Admin Only)
* Delete Products (Admin Only)

### Shopping Cart

* Add Items to Cart
* Update Product Quantity
* Remove Items from Cart
* Calculate Total Price

### Order Management

* Place Orders
* View Order History
* Automatically Clear Cart After Ordering

### Admin Features

* Admin Dashboard
* Manage Products
* Seller/Product Management

### Seller Request System

* Become Seller Page
* Users Can Request Seller Access

## ⚙️ Installation

### Clone Repository

```bash
git clone <repository-url>
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

Create a `.env` file:

```env
PORT=3000
DB_HOST=localhost
DB_PORT=5432
DB_USER=postgres
DB_PASSWORD=your_password
DB_NAME=ecommerce
JWT_SECRET=your_secret_key
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
